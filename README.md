## API Hackathon

### How to execute?
1. First, you need **Docker** and **Docker Compose** in your machine.

2. Clone repo
```sh
git clone https://github.com/mrbrunelli/hackathon-api.git
```

3. After repo cloned, enter in project folder
```sh
cd ./hackathon-api
```

4. Start containers
```sh
docker-compose up -d
```

5. Access application on
```sh
http://localhost:8085/
```
